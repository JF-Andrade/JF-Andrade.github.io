Classificador de Imagens de Flores

Um projeto de visão computacional que utiliza uma Rede Neural Convolucional (CNN) para identificar diferentes espécies de flores a partir de suas imagens.

O Problema de Negócio

O desafio foi construir um modelo que pudesse automatizar a identificação de flores, uma tarefa manual e demorada. O objetivo era criar uma solução robusta para ser usada em aplicações de botânica e jardinagem.

Ver no GitHub

Metodologia e Abordagem

  • **Coleta de Dados:** Utilizei o conjunto de dados "Flower Photos" do TensorFlow.
  • **Pré-processamento:** As imagens foram redimensionadas e normalizadas.
  • **Modelagem:** Construí uma CNN com camadas de convolução, pooling e dropout para extrair características das imagens.
  • **Treinamento:** O modelo foi treinado em 10 épocas, com validação cruzada para evitar overfitting.
  • **Ajuste Fino:** Realizei o ajuste de hiperparâmetros para otimizar a precisão.

Resultados e Conclusão

O modelo alcançou uma precisão de 98% no conjunto de dados de teste, demonstrando sua alta eficácia. A matriz de confusão abaixo (imagem) ilustra o desempenho. O notebook do projeto detalha os gráficos e as métricas de avaliação.
